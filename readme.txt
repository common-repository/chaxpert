== Installation ==

1. Extract "chaxpert-widget.zip".
1. Upload the "chaxpert" folder to your WordPress plugins directory.
1. Activate the "ChaXpert" from the WordPress plugins Admin Panel.
1. Update the options, from the Widgets Settings Page under ChaXpert on the Main Menu.
